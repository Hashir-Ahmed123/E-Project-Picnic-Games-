# 🌲 Picnic Games

**Picnic Games** is a modern front-end web project built for discovering **adventure games, resort getaways, picnic destinations, and curated outdoor itineraries** — all in one immersive platform.

Designed as an academic e-project with real-world inspiration, the platform combines travel discovery, activity booking concepts, and user authentication into a visually rich experience.

---

# ✨ Features

## 🏕 Resort Discovery

Explore scenic resort destinations with dedicated detail pages, highlights, and booking-inspired layouts.

* Featured luxury resorts
* Individual resort detail pages
* Responsive resort cards
* Visual destination galleries

## 🎯 Adventure Games

Browse and explore exciting outdoor adventure experiences such as:

* Archery
* Bungee Activities
* Jet Skiing
* Obstacle Courses
* River Rafting
* Scuba Adventures
* Water Slides
* Zorb Games

Each game includes its own detail page with imagery and descriptions.

## 🌍 Picnic Places

Discover curated outdoor destinations including:

* Beaches
* Mountains
* Waterfalls
* Camping Spots
* Farmhouses
* Waterparks
* Bonfire Locations

## 🗺 Custom Itineraries

Pre-designed outing plans for different trip styles:

* Beach trips
* Hiking adventures
* Camping experiences
* Waterpark itineraries

## 🔐 Authentication System

Firebase-powered authentication includes:

* User Sign Up
* Login System
* Google Sign-In
* User Dashboard
* Authentication State Handling
* Session Management

## 📱 Responsive UI

Built with a modern responsive layout featuring:

* Mobile navigation menu
* Interactive image galleries
* Modern card layouts
* Smooth user experience across devices

# 🚀 Getting Started

## 1 Clone Repository

```bash
git clone https://github.com/your-username/picnic-games.git
```

```bash

```

---

# 🔥 Firebase Setup

Update your Firebase credentials inside:

```javascript
firebase-config.js
```

Add your own Firebase configuration:

```javascript
const firebaseConfig = {
 apiKey: "YOUR_KEY",
 authDomain: "YOUR_DOMAIN",
 projectId: "YOUR_PROJECT_ID",
 storageBucket: "YOUR_BUCKET",
 messagingSenderId: "YOUR_SENDER_ID",
 appId: "YOUR_APP_ID"
};
```

Enable:

* Email/Password Auth
* Google Sign In

See `FIREBASE_SETUP.md` for full setup.
